1980's-Present

1983

TCP/IP

ARPANET adopted TCP/IP which would become the modern Internet.

1991

Internet Available

The internet was made available however, there wasn't much you could do at first.

1994

Netscape

One of the most dominant web browsers made available until Internet Explorer was released.

1995

Java Released

First appearing in 1995, it was designed to be a general-purpose programming language that was also intended to let developers "write once, run anywhere"

1998

Google Founded

Google specializes in internet-related services and their web browser Chrome is one of the most popular web browsers used today.

2004

Facebook

Launched on February 4th, 2004, it was social media website initially intended for Harvard students. Later it expanded to let anyone over the age of 13 use it and later became a billion dollar company.

2009

Windows 7

Released towards the end of 2009, it was a follow up to the poor critical reception of Windows Vista. It included a redesigned taskbar, new filing sharing features, and support for multitouch input.
